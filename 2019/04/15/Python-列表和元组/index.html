Python 列表和元组一

发表于 2019-04-15 | 分类于 Python基础 | 阅读次数：

在Python中，最基本的数据结构是序列，最常用的序列有列表、元组和字符串。在工作中，也是经常用到，本篇文章主要总结列表和元组的用法。

通用操作

有几种操作适用于所有序列，包括索引、切片、相加、相乘和成员资格检查

切片

除了使用索引外，Python提供了切片功能来访问特定范围内的元素，使用两个索引，并用冒号分隔

注意:提供的两个索引编号来指定切片的边界，第一个索引指定的元素包含在切片内，第二个则不包含。

切片的特殊用法

切片的步长
执行切片操作时，显式或隐式地指定起点和终点，但我们通常省略了另一个参数，即步长。在普通切片中，步长为1。这意味着从一个元素移到下一个元素，因此切片包含起点和终点之间的所有元素。

注意:步长不能为0!

成员资格

要检查特定的值是否包含在序列中，可使用运算符in。

len(),max(),min()

函数len返回序列包含的元素个数
min和max分别返回序列中最小和最大的元素

可变序列 列表

列表是一个可变序列，即可修改其内容。

列表方法

append

方法append用于将一个对象附加到列表末尾。

注意:append方法直接修改旧表，并且不返回值。

clear

clear清空列表。

copy

方法copy复制列表

count

方法count计算指定的元素在列表中出现了多少次。

extend

extend可以将多个值添加到列表末尾，换而言之，你可使用一个列表来扩展另一个列表。同append一样，extend直接修改了原列表，并不会返回值。

index

方法index在列表中查找指定值第一次出现的索引。

注意:当在列表中找不到指定的值时，index会引发异常。

insert

方法insert用于将一个对象插入列表。

pop

方法pop从列表中删除末尾元素，并返回这一元素。

remove

方法remove从列表中删除指定元素。同index一样，当找不到指定的值时，会引发异常

reverse

reverse按相反的顺序排列列表中的元素

使用list(reversed(numbers))可以返回一个相反顺序的列表。

sort

方法sort用于对列表直接排序。直接排序意味着对原来的列表直接进行修改。并且不返回值。
